# TrustLens

TrustLens is an AI-powered web application designed to detect fake news and analyze sentiments from textual content. The system helps users identify misleading information and understand whether the content expresses positive, negative, or neutral emotions.

## Features

* Fake News Detection
* Sentiment Analysis
* User Authentication
* Admin Dashboard
* Text Analysis Reports
* Secure Database Management
* Responsive User Interface

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript
* Bootstrap

### Backend

* Java
* Spring Boot

### Database

* MySQL

### Tools & Platforms

* VS Code
* MySQL Workbench
* Git & GitHub

## Modules

### User Module

* Register and Login
* Submit Text for Analysis
* View Sentiment Results
* Check Fake News Detection Reports

### Admin Module

* Manage Users
* Monitor Analysis Results
* Manage System Data

## Project Objective

The main objective of TrustLens is to reduce the spread of fake news and help users analyze online content effectively using AI-based techniques.
